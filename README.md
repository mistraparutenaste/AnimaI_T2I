# AnimaI T2I

画像生成AIのプロンプトをクリックだけで簡単に構築し、コピーできるツールです。
CSVファイルを編集することで、独自のタグを自由に追加・管理できます。

## 起動方法

### クリックで簡単に起動する（おすすめ）
- **`start.bat`** をダブルクリックしてください。
- 初回起動時はビルドが走るため少し時間がかかります。
- 自動的にブラウザが立ち上がり、アプリが開きます。

### コマンドで起動する
```bash
cargo run --release
```

## Rust未インストール環境で動かす
以下の手順で実行ファイルを作成できます。

1. **`build.bat`** をダブルクリックして実行します。
2. ビルドが完了すると、このフォルダの中に `animai.exe` という実行ファイルが作成されます。
3. 以下の2つを同じフォルダに入れて、セットにしてZIP等で固めて共有してください：
   - `animai.exe`
   - `csv` フォルダ



## CSVのカスタマイズ

このアプリは、`csv` フォルダの中にあるCSVファイルを起動時に自動的に読み込んでUI（ボタン）を生成します。

- 形式: `日本語名,英語プロンプト` (カンマ区切り)
- 例: `最高画質,"masterpiece, best quality"`
- `n_` から始まるファイル名（例：`n_pose.csv`）は、**NSFW用**のタグとして認識されます。
- `negative` という文字が含まれるファイル名（例：`negative_prompt.csv`）は、**ネガティブプロンプト用**のタグとして認識されます。
- 新しくCSVファイルを追加すると、自動的に新しいカテゴリとしてアプリの画面に追加されます。

## 依存関係・パスの仕様について
実行ファイル（EXE）を実行した際、システムは以下の優先順位で `csv` フォルダを探します。
1. 現在の作業ディレクトリ内の `csv` フォルダ
2. 実行ファイル本体と同じ階層にある `csv` フォルダ

これにより、ショートカットから起動した場合や、別のドライブから起動した場合でも、安全に `csv` フォルダを読み込むことができます。
